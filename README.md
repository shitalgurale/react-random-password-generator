# 🔐 Random Password Generator

A modern and responsive **Random Password Generator** built with **React.js** and **Bootstrap**. The application generates strong, customizable passwords using uppercase letters, lowercase letters, numbers, and special characters, with an option to copy the generated password to the clipboard instantly.

## ✨ Features

* 🔑 Generate secure random passwords
* 📏 Customize password length
* 🔠 Include uppercase letters (A–Z)
* 🔡 Include lowercase letters (a–z)
* 🔢 Include numbers (0–9)
* 🔣 Include special characters
* 📋 One-click copy to clipboard
* 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/random-password-generator.git
```

2. Navigate to the project folder:

```bash
cd random-password-generator
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```
